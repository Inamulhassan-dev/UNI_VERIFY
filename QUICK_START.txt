📋 3-STEP INSTALLATION
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Install Prerequisites (One-time only)
───────────────────────────────────────────────────────────────────────────────

  🐍 Python 3.8+
     Download: https://www.python.org/downloads/
     ⚠️ Check "Add Python to PATH" during installation!

  📦 Node.js 18+
     Download: https://nodejs.org/
     Choose LTS version


STEP 2: Extract & Setup (One-time only)
───────────────────────────────────────────────────────────────────────────────

  1. Extract UNI_VERIFY.zip to Desktop or USB drive
  
  2. Double-click: SETUP_FIRST_TIME.bat
  
  3. Wait 5-10 minutes for installation
  
  4. See "Setup Completed Successfully" ✅


STEP 3: Start Using
───────────────────────────────────────────────────────────────────────────────

  1. Double-click: START.bat
  
  2. Browser opens automatically
  
  3. Login with:
     📧 Email: admin@univerify.com
     🔑 Password: admin123

❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

  PROBLEM: "Python not found"
  ──────────────────────────────────────────────────────────────────────────
  ✅ Install Python from python.org
  ✅ Check "Add Python to PATH" during installation
  ✅ Restart computer and try again


  PROBLEM: "Node.js not found"
  ──────────────────────────────────────────────────────────────────────────
  ✅ Install Node.js from nodejs.org
  ✅ Restart computer and try again


  PROBLEM: "Failed to fetch" when logging in
  ──────────────────────────────────────────────────────────────────────────
  ✅ Make sure both terminal windows are open
  ✅ Run STOP.bat then START.bat again


  PROBLEM: Setup fails
  ──────────────────────────────────────────────────────────────────────────
  ✅ Check internet connection
  ✅ Right-click SETUP_FIRST_TIME.bat → "Run as administrator"
  ✅ Delete backend\.venv and frontend\node_modules folders, try again
